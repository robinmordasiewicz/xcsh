Add Labels to a configuration object

Usage:
  vesctl configuration add-labels [command]

Examples:
vesctl configuration add-labels virtual_host <name> --label-key acmecorp.com/attr-1 --label-value val-1

Flags:
  -h, --help                  help for add-labels
      --label-key strings     Key part of label
      --label-value strings   Value part of label
  -n, --namespace string      Namespace of configuration object (default "default")

Global Flags:
  -a, --cacert string         Server CA cert file path
  -c, --cert string           Client cert file path
      --config string         A configuration file to use for API gateway URL and credentials (default "/Users/r.mordasiewicz/.vesconfig")
      --hardwareKey           Use yubikey for TLS connection
  -k, --key string            Client key file path
      --outfmt string         Output format for command
  -o, --output string         Output dir for command (default "./")
      --p12-bundle string     Client P12 bundle (key+cert) file path. Any password for this file should be in environment variable VES_P12_PASSWORD
  -u, --server-urls strings   API endpoint URL (default [http://localhost:8001])
      --show-curl             Emit requests from program in CURL format
      --timeout int           Timeout (in seconds) for command to finish (default 5)

Use "vesctl configuration add-labels [command] --help" for more information about a command.
