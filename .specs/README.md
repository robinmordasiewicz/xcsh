## Quick Start

### Validate Specifications

```bash
# Using Spectral
spectral lint openapi.json

# Using swagger-cli
swagger-cli validate openapi.json
```

### Generate API Client

```bash
# Using OpenAPI Generator
openapi-generator-cli generate \
  -i openapi.json \
  -g python \
  -o ./generated-client

# Using Swagger Codegen
swagger-codegen generate \
  -i openapi.yaml \
  -l java \
  -o ./java-client
```
